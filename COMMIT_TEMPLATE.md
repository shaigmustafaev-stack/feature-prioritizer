# Commit Message Template

Используй один из вариантов:

```text
feat(scope): что добавлено
```

```text
fix(scope): что исправлено
```

```text
test(scope): какие тесты добавлены/изменены
```

Подсказки:
- `scope` должен быть коротким: `rice`, `rice-mobile`, `supabase`, `e2e`, `process`.
- Первая строка до ~72 символов.
- Без общих формулировок типа "changes", "updates", "fixes".
